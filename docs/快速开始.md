## 🎯 InStock 是什么？

**一句话概括**: InStock 是一个帮你自动抓取股票数据、计算技术指标、识别 K 线形态、执行选股策略的量化投资工具箱。

### 它能帮你做什么？

| 功能 | 说明 | 举个例子 🌰 |
|------|------|------------|
| 📥 数据抓取 | 自动获取 A 股每日行情 | 不用手动下载 Excel |
| 📈 指标计算 | 32 种技术指标一键生成 | MACD、KDJ、BOLL... |
| 🔍 形态识别 | 61 种 K 线形态自动识别 | 锤头、十字星、晨星... |
| 🎯 策略选股 | 11 种内置策略筛选股票 | 放量上涨、海龟交易... |
| ✅ 回测验证 | 验证策略历史胜率 | 看看策略靠不靠谱 |
| 🖥️ Web 展示 | 浏览器查看所有结果 | 手机、电脑都能看 |

### 🪟 Windows 用户极简安装

**Step 1: 安装 Python**

```bash
# 1. 下载 Python: https://www.python.org/downloads/
# 2. 安装时 ✅ 勾选 "Add Python to PATH"
# 3. 设置国内镜像 (解决下载慢的问题)
pip config --global set global.index-url https://mirrors.aliyun.com/pypi/simple/
```

**Step 2: 安装 MySQL**

```bash
# 下载地址: https://dev.mysql.com/downloads/mysql/
# 安装向导，一路下一步，记住你设置的密码!
```

**Step 3: 安装 TA-Lib**

```bash
# 下载地址: https://ta-lib.org/install/
# Windows 选择 "Windows Executable Installer"
```

> 💡 **小贴士**: 所有安装都选默认选项即可，不需要特殊配置。

## 第2步：安装配置 ⚙️

### 2.1 下载项目代码

```bash
# 方式一: Git 克隆 (推荐)
git clone https://github.com/myhhub/stock.git
cd stock

# 方式二: 直接下载 ZIP 包
# 去 GitHub 页面点击 "Code" → "Download ZIP"
```

### 2.2 安装依赖库

```bash
# 在项目根目录执行
pip install -r requirements.txt
```

> ⏳ 这一步需要几分钟，取决于你的网速

### 2.3 配置数据库连接

打开 `instock/lib/database.py`，修改数据库配置：

```python
db_host = "localhost"      # 数据库地址，本机就是 localhost
db_user = "root"           # 数据库用户名
db_password = "你的密码"    # 🔑 改成你安装 MySQL 时设置的密码
db_port = 3306             # 默认端口，一般不用改
```

> ⚠️ **重要**: 密码一定要改成你自己的！

## 第3步：运行验证 🎉

### 3.1 启动 Web 服务

**Windows:**
```bash
# 双击运行
instock\bin\run_web.bat
```

**Linux/Mac:**
```bash
bash instock/bin/run_web.sh
```

### 3.2 打开浏览器验证

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   🌐 打开浏览器，访问:                               │
│                                                     │
│        http://localhost:9988/                       │
│                                                     │
│   看到界面就说明成功啦！ 🎊                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 3.3 抓取第一批数据

```bash
# 方式一: 运行批处理 (Windows)
instock\bin\run_job.bat

# 方式二: 手动执行 Python 脚本
cd instock/job
python basic_data_daily_job.py
```

> ⏳ 首次运行需要几分钟，会自动创建数据库表

## 🆘 遇到问题？

### 常见问题速查表

| 问题 | 可能原因 | 解决办法 |
|------|----------|----------|
| `pip install` 失败 | 网络问题 | 检查是否配置了国内镜像 |
| 数据库连接失败 | 密码错误 | 检查 `database.py` 中的密码 |
| TA-Lib 导入错误 | 未安装 C 库 | 先安装 TA-Lib 头文件 |
| 端口被占用 | 9988 被其他程序使用 | 换个端口或关闭占用程序 |

> 📚 更多问题请查看 [FAQ 常见问题](./FAQ.md)
